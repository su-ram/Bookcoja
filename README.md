> 북코자
 
&nbsp;


&nbsp;청각장애인은 경보음, 재난 대피 방송 등 청각적 신호(소리)를 인지하기 어려워 화재의 초기 인지가 늦어지고, 언어장애를 동반한 경우가 많아 비장애인과 의사소통을 하는데 어려움이 있어 즉각적인 신고와 도움 요청이 힘들다. 따라서 이들을 위해 신속하게 화재 위험을 감지하여 알림과 동시에 안전한 탈출구를 안내하여 구조 요청까지 하는 일련의 특화된 과정이 필요하다. 이러한 문제의식을 바탕으로 **청각장애인을 위한 맞춤형 화재 대피 솔루션**을 기획하게 되었다. `“신속한 감지, 무조건 알림, 안전한 대피”`을 슬로건으로 하여 청각장애인이 화재 발생 시 빠르고 안전하게 대피하는 것을 목표로 하고 있다. 화재 사실을 신속하게 알림 받고 최적의 대피 경로를 통해 안전지대로 탈출할 수 있는 화재의 인지, 알림, 대피, 신고가 통합된 맞춤형 대피 솔루션을 제공한다. :fire:


```
HW
  ° 아두이노 (화재경보기, 진동침대, LED 무드등) 
  
SW
  ° 모바일 앱
  ° 웹 페이지
  ° LTE 통신
```

&nbsp;화재 경보기에서 여러 센서를 통해 화재를 감지하고, 웹 페이지와 모바일 앱은 현재 상황을 실시간으로 보여준다. 화재가 발생하면 SKT의 LTE 통신망을 통해 주변 기기들을 제어하여 화재 상황을 알리고 비상경로로 유도하여 대피할 수 있도록 한다. 

개발 기간 | 2018.03 ~ 2018.10

&nbsp;

##  :art:  사용 기술 Skill sets
- Mobile os : Android - Java
- Front End : HTML, Javascript, Ajax, Jquery
- Back End : php, Apache 
- Server os : Ubuntu
- DB : Mysql
- HW : Arduino - C 
- LTE : SKT Cat.M1
- Open API : ThingPlug
